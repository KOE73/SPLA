# SPLA Density Resources (Themes & Layout Keys)

This file describes all spacing, margin, padding, font, and corner radius resource keys used in `SPLA.UI.Avalonia` density themes. All density themes (`Max.axaml`, `Mini.axaml`, `Nano.axaml`, `Norm.axaml`) must define the same set of resources.

---

## 1. Layout Spacing

| Resource Key | Description |
| :--- | :--- |
| `StandardMargin` | The default outer margin for panels, blocks, and common containers. |
| `StandardPadding` | The default inner padding for cards, panes, and content containers. |
| `CompactMargin` | A tighter margin for dense layouts and nested containers. |
| `StandardSpacing` | The default spacing between controls inside a stack or grid. |

---

## 2. Chat Bubble Layout

| Resource Key | Description |
| :--- | :--- |
| `ChatBubbleMargin` | The base margin for chat bubbles in list and stream views. |
| `UserChatBubbleMargin` | The margin for user-authored chat bubbles, aligned toward the right side. |
| `AiChatBubbleMargin` | The margin for assistant/AI chat bubbles, aligned toward the left side. |

---

## 3. Typography

| Resource Key | Description |
| :--- | :--- |
| `HeaderFontSize` | The font size for section headers and prominent labels. |
| `BaseFontSize` | The default body font size for standard text. |
| `SmallFontSize` | The reduced font size for hints, metadata, and secondary text. |

---

## 4. Shape

| Resource Key | Description |
| :--- | :--- |
| `StandardCornerRadius` | The default corner radius for panels, cards, and controls. |
