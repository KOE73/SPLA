# Данные графа

`SPLA.Plugins.OneC` содержит UI-нейтральный слой подготовки ограниченного подграфа конфигурации 1С.

## Компоненты

- `Graph/OneCGraph.cs` — типизированные DTO узлов, связей, запроса и сводки результата;
- `Graph/OneCGraphBuilder.cs` — обход SQLite-индекса с ограничениями глубины и числа связей.

## Режимы

- `Dependencies` — исходящие зависимости объекта;
- `References` — входящие ссылки на объект;
- `DataFlow` — связи потоков данных: `writes`, `reads` и `queries`.

## UI

Готового графического представления сейчас нет. Старый Avalonia/WebView-слой удалён как недостижимый из текущей архитектуры.

Будущий web-клиент на Vue должен получать подготовленный подграф через сервисный или plugin-panel контракт. Выбор библиотеки визуализации и преобразование DTO в её формат принадлежат клиентскому слою; обход SQLite и ограничение результата остаются на стороне плагина.
