# Данные графа

`SPLA.Plugins.OneC` содержит UI-нейтральный слой подготовки ограниченного подграфа конфигурации 1С.

## Компоненты

- `Graph/OneCGraph.cs` — типизированные DTO узлов, связей, запроса и сводки результата;
- `Graph/OneCGraphBuilder.cs` — обход SQLite-индекса с ограничениями глубины и числа связей.

## Режимы

- `Dependencies` — исходящие зависимости объекта;
- `References` — входящие ссылки на объект;
- `DataFlow` — связи потоков данных: `writes`, `reads` и `queries`.

## UI

Плагин поставляет Vue-обозреватель через `web_settings_entry`. Сейчас он открывается как preview в `Settings → Plugins → onec` и показывает дерево объектов, поиск, карточку объекта и три режима графа.

Backend `Web/OneCWebActions.cs` преобразует результат `OneCGraphBuilder` в DTO и передаёт его клиенту через существующий `plugin.action`. Vue преобразует DTO в формат Cytoscape, а сама библиотека входит в plugin bundle и не загружается с CDN. Обход SQLite и ограничение результата остаются на стороне плагина.

Старый Avalonia/WebView/HTML-слой не возвращён. Для переноса preview из настроек в самостоятельную dock-панель потребуется только отдельная host-точка входа; контракт данных уже отделён от UI.
