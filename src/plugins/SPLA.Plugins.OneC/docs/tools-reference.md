# Инструменты агента

`SPLA.Plugins.OneC` предоставляет девять инструментов для индексации и анализа выгруженной конфигурации 1С.

## Индексация и поиск объектов

- `onec_build_index` — создаёт или инкрементально обновляет локальный SQLite-индекс по каталогу выгрузки;
- `onec_find_object` — ищет объекты по полному или частичному имени;
- `onec_get_object` — возвращает карточку конкретного объекта и связанные с ним данные;
- `onec_explain_object` — собирает структурированный контекст для объяснения назначения и использования объекта.

## Связи и зависимости

- `onec_find_references` — находит входящие ссылки на объект;
- `onec_get_dependencies` — возвращает исходящие зависимости объекта;
- `onec_get_reverse_dependencies` — возвращает объекты, зависящие от указанного;
- `onec_find_writers` — находит объекты, записывающие данные в указанный объект;
- `onec_find_readers` — находит объекты, читающие или запрашивающие данные указанного объекта.

Инструменты анализа работают с общим индексом `.spla/onec.sqlite`. Графические команды в текущем model-facing API отсутствуют; типизированная модель графа сохранена как внутренняя основа для будущего web-интерфейса.
