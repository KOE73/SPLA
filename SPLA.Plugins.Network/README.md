# SPLA Network Plugin

Network diagnostics and administration tools for SPLA.

## Responsible Use

SPLA may include network diagnostic and administration tools such as ping, DNS lookup, traceroute, HTTP checks, port checks, subnet discovery, TLS/certificate inspection, and similar utilities.

These tools are intended only for:

* diagnosing systems and networks you own or are explicitly authorized to administer;
* internal IT administration and troubleshooting;
* lab, educational, and research environments;
* security assessment performed with proper permission.

Do not use SPLA to scan, probe, attack, disrupt, bypass access controls, or collect information from third-party systems without authorization.

SPLA is designed as an assistant for legitimate engineering and administration work. The user is responsible for ensuring that any actions performed through the tool comply with applicable laws, contracts, organizational policies, and authorization boundaries.
